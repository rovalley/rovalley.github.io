State Websites Review

alaska.com

  • The blue, tan, and yellow color scheme complement each other nicely and the design is well done and simple to look at.
  • The website loads quickly.
  • The menu consists of a search bar and lists.
  • The website highlights important topics that can answer questions you might have.
  • The screen size is not fixed works well on a phone screen.

arkansas.com

  • The color scheme is good and the design is mostly pictures with text attached which is pleasant to look at.
  • The website loads quickly.
  • The menu has multiple lists and images with links attached.
  • The lists and images make it easy to navigate and there is a chat box to help get anwsers on what you are looking for.
  • The screen is a fixed size cannot see everything on a phone screen.

Least favorite state website

kansas.com

  • The color scheme does not catch the eye but the design of the background mural does.
  • The website loads quickly.
  • The menu is only a searchbar.
  • There is text rotating through the searchbar of possible questions but overall it is confusing to know what to ask.
  • The screen is a fixed size zoomed in on a phone screen.
  • This is my least favorite website because there should be more than just a searchbar and a description just saying get answers.

Favorite state website

maine.com

  • The color scheme is simply white but uses an eye appealing image for the background design.
  • The website loads quickly.
  • The menu consists of a search bar and tabs.
  • The tabs are easy to navigate and the homepage has important general information to find what you are looking for.
  • The screen is fixed vertically but able to view everything easily on a phone screen.
  • This is my favorite website because it has the best home screen and it is the easiest to navigate.

maryland.com

  • The color scheme is white and red but the red covers the background image design which does not look good.
  • The website loads quickly.
  • The menu has a search bar, tabs and images with links attached.
  • All three of the menu options make the website easy to navigate or find what you are looking for.
  • The screen is fixed vertically and a little more difficult to view everything on a phone screen.
